The flexible box layout mode, makes it easier to design flexible resposive layout structure without having to use floats or positioning

A flexbox layout consists flex-container with one or more flex-items

A flex-container becomes flexible by setting display property to flex

1
2
3

Flexbox properties for the flex-container are

  • Flex-direction
  • Flex-flow
  • Flex-wrap
  • justify-content
  • align-items
  • align-content
